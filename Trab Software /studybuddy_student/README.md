# StudyBuddy - Projeto Estudante (Expo)

Este é um projeto exemplo feito com aparência de código escrito por um aluno.
Ele contém telas mínimas: Splash, Main, Tasks, About e um serviço simples de DB usando expo-sqlite.

Como rodar:
1. Instale o Expo CLI: `npm install -g expo-cli` (se precisar).
2. `npm install`
3. `expo start`
4. Abra em Expo Go ou emulador.

Observação: Este projeto é um template educativo. Ajustes podem ser necessários dependendo da versão das libs.
